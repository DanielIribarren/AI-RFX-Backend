1.1 Principio Central: La Inteligencia Está en el Agente

No creamos lógica estática para cada operación. Creamos UN agente de IA inteligente con un system prompt robusto que maneja TODOS los casos.

1.2 Por Qué Este Enfoque es Superior
Ventajas:

Flexibilidad: Cambiar comportamiento = cambiar prompt, no código
Escalabilidad: Agregar nuevos casos = extender prompt, no crear servicios nuevos
Mantenibilidad: Un solo archivo de prompt vs múltiples servicios
Inteligencia Natural: LLM entiende contexto y matices mejor que código rígido
Mejora Continua: Refinar prompt es más rápido que refactorizar código

El System Prompt es el Cerebro:

90% de la lógica está en el prompt
10% es código simple de ejecución
